To view and interact with your new blog, please follow these steps:

### 1. Install Dependencies

Open your terminal in the project root and run the following command to install all the necessary packages for the application:

```bash
npm install
```

### 2. Run the Development Server

After the installation is complete, start the local development server with this command:

```bash
npm run dev
```

This will launch the application. Your terminal will display a local URL (typically `http://localhost:5173`). Open this URL in your web browser to see your new blog live.
