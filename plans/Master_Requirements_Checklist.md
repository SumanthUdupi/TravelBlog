# Master Requirements Checklist

## 1. Content Integration (REQ-CI)
- [ ] **REQ-CI-001**: Implement Blog Cards with featured image, title, metadata, excerpt, and link.
- [ ] **REQ-CI-002**: Implement Scrollytelling sections with parallax background and animated text reveal.
- [ ] **REQ-CI-003**: Implement Parallax Sections for Hero and transitions (50% scroll speed differential).
- [ ] **REQ-CI-004**: Implement Timeline Layouts (Vertical/Horizontal) with day markers and expandable sections.
- [ ] **REQ-CI-005**: Adhere to Typography Hierarchy (Cinzel, Playfair Display, Lato).
- [ ] **REQ-CI-006**: Ensure Accessibility Standards (Contrast 4.5:1, Alt text, Keyboard Nav).
- [ ] **REQ-CI-007**: Implement Interactive Media (360 views placeholder, Audio players placeholder).

## 2. Interactive Experience (REQ-IX)
- [ ] **REQ-IX-001**: Implement Hover Effects on Cards (Elevation, Shadow).
- [ ] **REQ-IX-002**: Implement Interactive Map (SVG based, Zoom/Pan, Tooltips).
- [ ] **REQ-IX-003**: Implement Media Galleries with Lightbox functionality.
- [ ] **REQ-IX-004**: Implement Reading Progress Indicator.
- [ ] **REQ-IX-005**: Implement Dark Mode Toggle (consistent with existing site).
- [ ] **REQ-IX-006**: Implement Smooth Scroll for all anchor links.
- [ ] **REQ-IX-007**: Implement "Back to Top" functionality.

## 3. Immersive Design (REQ-ID)
- [ ] **REQ-ID-001**: Implement Fade-in/Reveal animations on scroll (Intersection Observer).
- [ ] **REQ-ID-002**: Implement Atmospheric Backgrounds (Gradients, Patterns).
- [ ] **REQ-ID-003**: Implement Micro-interactions (Button states, Link hovers).
- [ ] **REQ-ID-004**: Responsive Design (Mobile, Tablet, Desktop layouts).
- [ ] **REQ-ID-005**: Loading States (Skeleton screens or Spinners for dynamic content).

## 4. Technical Specifications (REQ-TS)
- [ ] **REQ-TS-001**: Use Semantic HTML5 (header, nav, main, article, footer).
- [ ] **REQ-TS-002**: Use BEM/Modular CSS naming convention.
- [ ] **REQ-TS-003**: No external frameworks (React, Vue, jQuery) - Vanilla JS only.
- [ ] **REQ-TS-004**: Performance Optimization (Lazy loading images).
- [ ] **REQ-TS-005**: Cross-Browser Compatibility (Chrome, Firefox, Safari, Edge).

## 5. Visual Consistency (REQ-VC)
- [ ] **REQ-VC-001**: Match Color Palette exactly (Gold: #c5a059, Brown: #8b4513, etc.).
- [ ] **REQ-VC-002**: Match Font Family usage exactly (Cinzel for Headers, Lato for Body).
- [ ] **REQ-VC-003**: Reuse existing Header/Footer structure.
